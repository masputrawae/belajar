Apa saja yang sedang saya pelajari

Belajar itu seru, walaupun terkadang bikin pusing. Tapi dengan semangat, kesabaran, kerja keras, dan doa, semua pasti bisa dipelajari. Ini dia beberapa hal yang sedang saya pelajari sekarang:

HTML & CSS

HTML & CSS itu seperti seni, ibarat kanvas dan cat. Saya suka bagaimana elemen-elemen kecil bisa berubah menjadi website yang cantik dan menarik!

Javascript

JavaScript? Wah, ini tantangan besar buat saya. Rasanya rumit, tapi saya yakin, dengan latihan dan doa, suatu hari nanti saya akan menguasainya. 😅

PHP

PHP bikin saya penasaran. Mungkin belum se-populer dulu, tapi siapa tahu ini akan jadi senjata rahasia saya di dunia back-end development!

Python

Python itu katanya mudah dipelajari. Saya anggap ini seperti ular yang harus dijinakkan. Sedikit usaha, dan semoga saya bisa menjadi pawangnya. 🐍

Bootstrap & Tailwind

Framework ini seperti shortcut untuk bikin desain responsif. Tapi saya masih penasaran, apakah ini benar-benar mempermudah atau justru bikin saya malas? Hehe.

Linux

Linux si penguin ini sudah jadi teman coding saya. Sekarang cuma sekadar pakai, tapi suatu saat saya ingin mendalami lebih jauh kemampuannya.

Jekyll

Jekyll adalah alat favorit saya untuk membuat website statis. Cepat, sederhana, dan sangat efektif. Saya suka mempelajarinya lebih dalam.

Ilmu copy paste

Siapa yang tidak suka copy-paste? Teknik ini memang menyelamatkan hidup saya, meskipun kadang bikin pusing kalau kode yang dipaste error. 😅
