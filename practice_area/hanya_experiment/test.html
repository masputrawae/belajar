Pendahuluan

Belajar web development itu menyenangkan! Anda bisa membangun website sesuai kebutuhan Anda. Berikut adalah beberapa poin penting:

  • Belajar HTML untuk struktur.
  • CSS untuk styling.
  • JavaScript untuk interaktivitas.

HTML dan CSS

HTML adalah dasar dari sebuah website. Dengan CSS, Anda bisa membuatnya lebih menarik:

“HTML memberikan kerangka, CSS memberikan keindahan.”

Contoh kode:

<html>
    <head>
        <title>Website Saya</title>
    </head>
    <body>
        <h1>Selamat Datang</h1>
    </body>
</html>
